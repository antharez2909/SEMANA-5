# Galería de Imágenes Interactiva

Proyecto realizado con HTML, CSS y JavaScript para manipular el DOM y manejar eventos.

## Funcionalidades
- Agregar imágenes usando una URL
- Seleccionar imágenes con clic
- Eliminar la imagen seleccionada
- Atajos de teclado:
  - Enter: agregar imagen
  - Delete: eliminar imagen seleccionada

## Tecnologías
- HTML5
- CSS3
- JavaScript (DOM)
